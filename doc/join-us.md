# 加入我们

**NexCore** 是一个充满活力与技术热情的团队。如果你对代码有洁癖，对技术有追求，渴望在实际项目中打磨自己的技能，欢迎加入我们！

## 🎯 我们正在寻找

### 1. 前后端开发工程师 (Full Stack Developer)

**你将负责：**
- 参与企业级 Web 应用的全栈开发 (Vue3 + Python/Node.js)。
- 设计并实现高可用的 RESTful API。
- 持续优化前端性能与用户体验。

**我们需要你：**
- 熟练掌握 **Vue 3** 生态及现代前端工程化工具。
- 熟悉至少一门后端语言 (**Python/Flask** 或 Node.js)。
- 理解数据库设计 (MongoDB/MySQL) 与基本查询优化。
- 有良好的代码规范意识，习惯写注释与文档。

### 2. AI 算法工程师 (AI Engineer)

**你将负责：**
- 负责业务场景下的模型选型、训练与调优。
- 将算法模型封装为高性能的推理服务 (API)。
- 探索 LLM (大语言模型) 在实际业务中的应用。

**我们需要你：**
- 扎实的机器学习/深度学习理论基础。
- 熟练使用 **PyTorch** 或 TensorFlow。
- 熟悉常见的 NLP 或 CV 模型架构。
- 具备良好的工程落地能力，不只是在 Notebook 里跑模型。
