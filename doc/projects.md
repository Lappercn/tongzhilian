# 项目案例

NexCore 团队在多个领域积累了丰富的实战经验，从企业级应用到 AI 算法落地，我们始终致力于交付高质量的解决方案。

## 💼 企业级应用

### 1. 全球化内容管理平台 (CMS)

为某出海企业打造的多语言内容管理系统，支持全球内容分发与 SEO 优化。

- **挑战**: 需要支持 10+ 种语言，且对首屏加载速度 (LCP) 要求极高。
- **解决方案**:
    - 采用 **SSR (服务端渲染)** 技术提升 SEO 与首屏性能。
    - 设计了灵活的 **动态表单引擎**，运营人员可自定义页面结构。
    - 集成 **CDN 全球加速** 与图片自动压缩服务。
- **成果**: 页面加载速度提升 60%，Google 收录量增长 200%。

### 2. 工业物联网数据可视化大屏

面向制造业的实时生产监控大屏，展示设备状态、产量统计与异常告警。

- **技术栈**: Vue3, ECharts, WebSocket, MQTT
- **亮点**:
    - **实时性**: 通过 WebSocket 实现毫秒级数据推送。
    - **自适应**: 独创的缩放算法，完美适配各种分辨率的大屏设备。
    - **高性能**: 针对海量数据点进行 Canvas 渲染优化，保持 60fps 流畅度。

## 🤖 AI 算法落地

### 1. 多鱼理财 (DuoYu Finance) - AI 多智能体投顾系统

基于 AI Agent 技术的下一代智能投顾系统，模拟顶级基金公司的决策流程。

- **场景**: 智能股票分析、多视角逻辑博弈、实时行情解读。
- **亮点**:
    - **多智能体辩论**: 多头、空头、主持人多视角博弈。
    - **实时数据驱动**: 接入实时行情与新闻，拒绝幻觉。
    - **3D 沉浸体验**: 虚拟数字人播报与可视化报表。
- **链接**: [查看项目详情](/projects/duoyu) | [在线体验](https://tongzhilian.cn/duoyu/)

### 2. 智能客服问答系统 (RAG)

基于大语言模型的企业内部知识库问答助手。

- **场景**: 帮助员工快速查询内部文档、报销流程与技术规范。
- **方案**:
    - **RAG 架构**: 结合 LangChain + Vector DB (向量数据库)。
    - **文档处理**: 自动解析 PDF/Word/Markdown，进行分块与向量化。
    - **混合检索**: 结合关键词检索与语义检索，提升召回准确率。
- **效果**: 问题解决率达 85%，大幅降低人工客服压力。

### 3. 电商个性化推荐引擎

为电商平台提供的商品推荐服务，提升转化率。

- **模型**: DeepFM + 序列推荐模型。
- **工程**:
    - 构建了 **实时特征工程** 流水线。
    - 实现了 **A/B Test** 框架，快速验证模型效果。
- **指标**: 点击率 (CTR) 提升 15%，GMV 增长 8%。
